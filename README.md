# Automation

В этом репозитории - отработка практических навыком автоматизации:

- поиск элементов на странице и взаимодействие с ними
- валидация данных
- работа с модальными окнами
- прокрутка страницы
- использование явных ожидания для стабильности теста

Репозиторий будет пополняться в процессе получения новых знаний
